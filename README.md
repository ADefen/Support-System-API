# Support-System-API
Мини-проект на FastAPI + SQLAlchemy.
Демонстрирует работу с реляционной БД, валидацию данных, разделение слоёв приложения и стандартные паттерны FastAPI.

Проект реализует API для системы поддержки: регистрация пользователей, создание заявок (тиккетов), управление статусами заявок, получение списков.

## Стек технологий

Python 3.13+

FastAPI — асинхронный веб-фреймворк.

SQLAlchemy 2.0+ — ORM для работы с БД.

SQLite — база данных (для быстрого старта).

Pydantic 2+ — валидация данных и сериализация.

email-validator — валидация email-адресов.

python-dotenv — управление переменными окружения.

## Возможности

Регистрация пользователей (POST /users) с проверкой уникальности username.

Создание заявок (POST /tickets) с привязкой к пользователю.

Получение информации о пользователе (GET /users/{user_id}).

Просмотр всех заявок пользователя (GET /tickets/owner/{owner_id}).

Изменение статуса заявки (PATCH /tickets/{ticket_id}/status).

Удаление заявки (DELETE /tickets/{ticket_id}).

Автоматическая валидация полей (длина username, формат email, допустимые статусы).

Swagger UI для тестирования API: http://127.0.0.1:8000/docs.

## Запуск

### Создай и активируй виртуальное окружение:

python -m venv venv

# Windows:
venv\Scripts\activate

# macOS/Linux:
source venv/bin/activate

### Установи зависимости:

pip install -r requirements.txt

### Настрой переменные окружения:

Создай файл .env в корне проекта и добавь туда:

DATABASE_URL=sqlite:///./support.db

### Запусти сервер:

uvicorn app.main:app --reload

### Открой в браузере: http://127.0.0.1:8000/docs — там будет интерактивная документация со всеми эндпоинтами.
